Contoh Project Godot Engine 3 - Nav Mesh Agent Sederhana

RAKIFSUL November 27, 2023 Premium

Godot Engine 3 adalah game engine yang cukup lengkap fiturnya.

Salah satu kelengkapannya dibuktikan dengan adanya fitur pathfinding.

Fitur pathfinding tersebut diwujudkan dengan penggunaan Navigation Mesh atau  NavMesh.

Sayangnya, fitur tersebut belum dilengkapi dengan agent-nya.

Jadi, belum ada NavMeshAgent di Godot Engine versi 3.4, setahu saya, saat tulisan ini dibuat.

Oleh karena itu, saya mencoba mencari solusinya.

Caranya dengan membuat script NavMeshAgent buatan saya sendiri.

Script Godot Engine ini menggunakan fitur pathfinding dari fitur NavMesh yang disediakan oleh Godot Engine.

Jika sampai di sini Anda belum paham dengan kata-kata saya  tadi, sebenarnya NavMeshAgent yang saya buat ini kurang lebih mirip dengan yang ada di Unity.

Hanya saja ini buatan sendiri.

Dalam project ini, saya membuat karakter capsule sederhana yang ditempatkan di atas plane.

Di plane ada obstacle-nya berupa cube.

Saat NavMesh sudah di-generate, maka pathfinding bisa dilakukan.

NavMeshAgent saya kemudian mencari path, kemudian berjalan melaluinya sebisa mungkin sambil menghindari obstacle.

NavMeshAgent ini bekerja dengan mengubah input posisi target menjadi array posisi path node.

Adapun posisi target diperoleh dengan mengubah posisi kursor di koordinat layar menjadi posisi di ruang 3D.

Manfaat dari script ini adalah untuk memberikan gambaran kepada Anda tentang bagaimana memanfaatkan path yang ditemukan dari pathfinding dan menerapkannya pada agent.

Disclaimer:

  • Source code hanya teruji di Windows 11 dan tidak dijamin akan berjalan dengan benar.
  • Source code hanya teruji di Godot Engine 3.5.2 dan tidak dijamin bisa berjalan dengan benar.
